🎰 E-services Tirage

Suppression des données

E-services Kouba · e.services.kouba@gmail.com

E-services Tirage ne conserve pas de base de données de vos commentaires ou publications. Les données sont récupérées temporairement pour effectuer le tirage, puis ne sont pas stockées.

Ce qui est conservé

  • Uniquement votre jeton de connexion, dans votre session (cookie), supprimé dès que vous vous déconnectez ou que la session expire.

Comment supprimer vos données / révoquer l'accès

  1. Dans l'application : cliquez sur « Déconnecter » — votre jeton de session est immédiatement supprimé.
  2. Depuis Facebook : Paramètres → Applications et sites web → sélectionnez « TirageConcours » → Supprimer.
  3. Par email : écrivez à e.services.kouba@gmail.com avec l'objet « Suppression des données ». Nous confirmons sous 7 jours.
